Wikify - Build wiki from my notes

build jar file:

mvn package # target/wikify.jar

run jar file

java -jar target/wikify.jar
# you can specify input/output dir by "-i <DIR>"/"--input <DIR>", "-o <DIR>"/"--output <DIR>".
